# 3.6 Тестирование программного продукта

Тестирование разработанного программного обеспечения является обязательным этапом, подтверждающим соответствие продукта требованиям технического задания. Целью испытаний является проверка корректности выполнения всех заявленных функций системы «Умный Коворкинг» в различных сценариях использования, оценка стабильности работы и надёжности механизмов обработки исключительных ситуаций.

Тестирование проводилось по методу «чёрного ящика» (Black Box Testing) на основе разработанных тест-кейсов (Test Cases). Дополнительно выполнялся регрессионный прогон автоматизированных тестов pytest. Общие сведения о проведении испытаний представлены в таблице 3.
